# 🔄 たてなおし

**崩れた勉強計画を自動で引き直す、大学の学期用スタディプランナー。**

計画アプリが挫折する最大の理由——「初日に立てた計画が3日目に現実と合わなくなり、直すのが面倒で開かなくなる」——だけを解決します。記録のアプリ（Studyplus等）とは土俵が違う、**再計画のアプリ**です。

## 特徴

- **再計画はボタンではなく常時**: 開いた瞬間に、昨日までの未消化が吸収された「今日やること」が出ている。繰越ボタンはそもそも存在しない
- **サボり前提**: 「今日は無理 🫠」ワンタップで今日の分を明日以降へ配り直し
- **正直**: 間に合わない締切は誤魔化さず警告し、何を削るべきかまで提示
- **大学の学期構造が一級市民**: 科目・試験・レポート・小テスト・曜日ごとの可処分時間
- **完全ローカル**: アカウント不要・サーバーなし。データは端末のlocalStorageのみ。JSONでバックアップ/引っ越し
- **AIはおまけで本体はアルゴリズム**: Gemini（無料枠・任意）に状態を渡して「引き直して」と相談できるが、キーなしでも全機能が動く

## 使い方

`index.html` をブラウザで開くだけ。科目名と締切日を入れれば、その日から「今日やること」の計算が始まります（タスク分割はあとからで大丈夫）。

## 開発

- 再配分エンジン（EDF貪欲割当＋余裕度警告）は純関数として `index.html` 内のマーカーブロックに隔離
- テスト: `node tests/planner.test.mjs`

## アーキテクチャの芯

状態（JSON）とエンジン（再配分の純関数）を分離し、AIは「状態を読んで提案するだけの交換可能なエンジン」に徹する設計。AIの応答はスキーマ検証→許可フィールド制限→差分プレビュー承認の3段ガードを通り、タスクの完了状態や締切日は書き換えられません。
